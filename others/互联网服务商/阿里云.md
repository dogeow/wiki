## 初始化小于等于2 TiB数据盘（Linux）

```
fdisk -u /dev/vdb
# 按 p，然后一直默认，最后按 w。

mkfs -t ext4 /dev/vdb1

cp /etc/fstab /etc/fstab.bak

echo `blkid /dev/vdb1 | awk '{print $2}' | sed 's/\"//g'` /www ext4 defaults 0 0 >> /etc/fstab

mount /dev/vdb1 /www

df -lh
```
